# Hosts

This folder contains configurations for different machines.

## Naming scheme
The folder names must match the expected hostnames. Every hostname must match `([a-z]{3})-([a-z]{3,})` or in other words, it is composed by a 3 letter prefix, a hyphen and a name consisting of 3 or more lowercase ASCII letters.

### Prefixes
Currently the following prefixes are used
- `lab` for the lab network
- `mob` for mobile devices
- `ovh` for machines hosted at OVH
- `vms` for virtual machines
